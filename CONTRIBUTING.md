# Contributing

Thank you for helping improve this project.

This repository exists to make immigrant visa preparation guidance easier to understand, easier to maintain, and safer to reuse.

## Before contributing
Please read:
- `README.md`
- `DISCLAIMER.md`
- `MAINTAINED_SOURCE_POLICY.md`
- `docs/ops/TRUST_AND_SAFETY.md`

## What kinds of contributions are helpful
- bug fixes in code or build scaffolding
- accessibility improvements
- bilingual clarity improvements
- source-backed content corrections
- rule-logic fixes tied to reliable sources
- issue reports when official instructions appear to have changed

## Content standard
For content and structured guidance changes:
- prefer official sources where available
- distinguish official requirements from practical advice
- note when a rule is post-specific or case-specific
- avoid overstating certainty on edge cases
- preserve bilingual consistency where applicable

## High-risk topics
Changes involving waivers, inadmissibility, criminal history, prior immigration violations, or other legally sensitive subjects may receive extra scrutiny or be declined unless they are tightly grounded in reliable sources and clearly framed.

## Licensing of contributions
By submitting a contribution, you agree that:
- code contributions are licensed under Apache 2.0
- editorial and guidance content contributions are licensed under CC BY-SA 4.0
- brand rights are not granted through contribution

## Attribution and changes
When making significant content changes, include:
- what changed
- why it changed
- the source or rationale used
- whether the change affects one post, one case type, or multiple workflows
