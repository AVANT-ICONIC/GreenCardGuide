# Bilingual Strategy

## Principle
English and Spanish are first-class product surfaces.

## Recommendations
- Store stable content keys once, with localized copy per language.
- Maintain a bilingual glossary authority.
- Review Spanish for naturalness, not just correctness.
- Use real user language where appropriate alongside formal terms.

## Example terminology map
- petitioner / patrocinador o peticionario
- beneficiary / beneficiario
- affidavit of support / declaracion jurada de patrocinio economico
- civil document / documento civil
- derivative / derivado
- appointment letter / carta de cita

## UX requirements
- language switch persists across routes
- print views honor selected language
- checklist outputs are fully localized
