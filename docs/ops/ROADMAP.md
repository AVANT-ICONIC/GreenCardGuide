# Roadmap

## Stage 0: Research and validation
- interview recent applicants and sponsors
- gather real confusion and failure cases
- inventory official sources
- build document taxonomy
- define trust/risk policy

## Stage 1: Foundation
- define data model
- define content types
- seed glossary
- create source registry
- build first rules matrix for Ciudad Juárez family-based cases

## Stage 2: MVP
- launch bilingual home page
- launch Ciudad Juárez prep hub
- launch checklist builder
- launch print mode
- launch common mistakes page
- launch feedback capture

## Stage 3: Improve case intelligence
- joint sponsor logic
- derivative child logic
- prior marriage logic
- stronger financial packet guidance
- sponsor mode vs applicant mode

## Stage 4: Internal operations tooling
- source review dashboard
- review queue
- content diffs
- publish controls
- rule audit views

## Stage 5: Expansion
- more posts
- more visa categories
- additional languages if needed
- constrained assistant over approved content
