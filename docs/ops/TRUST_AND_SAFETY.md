# Trust and Safety

## Product posture
This product provides informational preparation guidance. It is not legal advice and should not claim to replace official instructions.

## Statement classes
Every statement should fit one category:
- official requirement
- official recommendation
- common practical advice
- anecdotal field note
- legal nuance requiring caution

## High-risk topics
These should be carefully limited and/or routed to professional help:
- inadmissibility grounds
- criminal history
- waivers
- prior immigration violations
- custody/adoption edge cases
- medical inadmissibility
- document unavailability exceptions

## Operational requirements
- every checklist rule traces to a source or approved editorial policy
- every content page has last reviewed date
- uncertain pages can be flagged for re-review
- source changes create review tasks
