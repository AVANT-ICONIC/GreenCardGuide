# Source Registry Notes

This directory is for source metadata snapshots, review notes, and future change-detection artifacts.

Recommended structure:
- source index
- manual review notes
- snapshot metadata
- diffs / change logs

Current assumptions:
- `source-references.json` is the first typed registry stub for source metadata.
- The current entries are project-governance references used to explain placeholder trust state, not official immigration requirement sources.
- Placeholder content loaders may attach these references consistently until source-backed public guidance is added.

Do not treat community anecdotes as equivalent to official requirements.
